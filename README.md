# NexShell

A personal PowerShell configuration with utilities and features.

## English

### Installation

Run this command in PowerShell:
```powershell
irm https://raw.githubusercontent.com/aksiez/PowerShell/main/install.ps1 | iex
```

### Features

- **upd** - Update NexShell to the latest version
- **reload** - Reload your profile
- **batt** - Show battery status
- **ip** - Show your public IP
- **port** - Check what's using a specific port
- **psx** - Show top processes by CPU
- And more!

### Configuration

Edit `config.toml` in your NexShell folder:
- `language` - Set to `english` or `franĂ§ais`
- `showWelcome` - Set to `false` to disable the welcome message
